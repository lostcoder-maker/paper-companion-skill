# Lawful Paper Sources

Use this reference when searching, downloading, or explaining why a source is allowed.

## Strong Defaults

- **arXiv**: official API and PDF URLs. Good for CS, math, physics, quantitative fields.
- **PubMed Central**: open biomedical full text. Prefer official article and PDF links.
- **Institutional repositories**: university-hosted author manuscripts when publicly offered.
- **Publisher open-access pages**: download only when the page marks the article as open access or provides a public PDF.

## Metadata / Discovery

- **Crossref**: DOI metadata. Not a universal PDF source.
- **OpenAlex**: open metadata and OA location discovery.
- **Semantic Scholar**: metadata, abstracts, citation context, sometimes open PDF URLs.

## Avoid

- Paywall bypasses, Sci-Hub mirrors, token/cookie replay, private library credentials, or any source that is not authorized to distribute the paper.

## Practical Rule

If a paper has no lawful PDF, provide DOI, abstract, metadata, publisher page, and ask the user to supply an authorized local copy for indexing.
